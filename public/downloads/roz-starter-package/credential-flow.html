Roz starter package

OAuth-first setup flow

This local page demonstrates the intended setup flow. For the small setup app, run python3 apps/roz_setup_app.py after unzipping the package.

1. ChatGPT
Sign in with the ChatGPT/OpenAI account you already use.
2. Google
Use OAuth for Gmail, Calendar, and Drive access.
3. Telegram
Message the prebuilt Roz onboarding bot with your setup code.

Security rule: the setup code verifies purchase; it is not an API token. Never include real credentials in screenshots, support requests, website source, package docs, or Shopify Digital Downloads descriptions. Use [REDACTED] in examples.
